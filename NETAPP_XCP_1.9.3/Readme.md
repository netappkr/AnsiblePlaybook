# XCP scan
ontap api만으로는 볼륨 내부 file 레벨의 정보를 얻기가 어렵습니다.</br>
Ansible로 XCP 명령을 실행하여 File 레벨 정보를 얻어와 작업합니다.</br>
ontap에서 file_analytics 를 활성화 하면 rest api를 활용하여 데이터를 가져올 수 있습니다.
